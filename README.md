# SOS Animals – Hackathon Project

**Helping abandoned animals find the care they need through real-time reporting and NGO coordination.**

-----

## 🚨 The Problem

Every year, millions of animals are abandoned, yet NGOs often struggle to track and rescue them effectively due to the lack of a centralized reporting and response system.

## ✅ The Solution

The **SOS Animals App** provides a streamlined platform for real-time reporting. By using integrated maps and intuitive forms, the app simplifies the reporting process and automatically notifies the nearest NGO to ensure a swift response. 

-----

## 🔹 Features

**Location Tracking:** Integration with **Mapbox API** for precise incident reporting.
**Multi-step Reporting Form:** A user-friendly, guided process for submitting animal alerts.
**User Authentication:** Secure access managed via **Clerk**.
**NGO Alerts:** Automated notification system to bridge the gap between reporters and rescuers.

## ⚙️ Tech Stack

 **Frontend:** ReactJS, CSS
**Authentication:** Clerk
**Deployment:** Vercel, Netlify
**Maps:** Mapbox API

-----

## 🛠️ Challenges & Solutions

During the hackathon, we overcame several technical hurdles:

**Authentication:** Resolved login flow issues by improving the authentication logic.
**Deployment:** Fixed deployment errors by debugging API access.
**Mapping:** Addressed map rendering issues by optimizing the Mapbox API integration.

-----
