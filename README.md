# SOS Animals – Hackathon Project

[cite_start]**Helping abandoned animals find the care they need through real-time reporting and NGO coordination.** [cite: 1, 2]

## 🚨 The Problem

[cite_start]Every year, millions of animals are abandoned, yet NGOs often struggle to track and rescue them effectively due to the lack of a centralized reporting and response system. [cite: 5, 6, 7, 8]

## ✅ The Solution

[cite_start]The **SOS Animals App** provides a streamlined platform for real-time reporting. [cite: 9, 10] [cite_start]By using integrated maps and intuitive forms, the app simplifies the reporting process and automatically notifies the nearest NGO to ensure a swift response. [cite: 11, 12]

-----

## 🔹 Features

  * [cite_start]**Location Tracking:** Integration with **Mapbox API** for precise incident reporting. [cite: 15]
  * [cite_start]**Multi-step Reporting Form:** A user-friendly, guided process for submitting animal alerts. [cite: 16]
  * [cite_start]**User Authentication:** Secure access managed via **Clerk**. [cite: 17]
  * [cite_start]**NGO Alerts:** Automated notification system to bridge the gap between reporters and rescuers. [cite: 18]
